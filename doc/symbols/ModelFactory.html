  • registerModels (models)
    注册APP中用到的model
    Parameters
    •   models {Object|Array} 模块描述信息
    •   models.type {String} app中model的唯一标识
    •   models.ops {Object} 传递的参数信息，如{uri:'test',isJSONP:true,updateIdent:true}
    •   models.gets {Object} 发起请求时，默认的get参数对象
    •   models.posts {Object} 发起请求时，默认的post参数对象
    •   models.cacheKey {String} 指定model缓存的key，当指定后，该model会进行缓存，下次不再发起请求，可使用该key通过getIf方法获取缓存的这个model
    •   models.expires {Integer} 缓存过期时间，以毫秒为单位，当过期后，再次使用该model时会发起新的请求(前提是该model指定cacheKey被缓存后expires才有效)
    •   models.before {Function} model在发起请求前的回调
    •   models.after {Function} model在发起请求，并且通过Model.sync调用success后的回调
    Example
    KISSY.add("app/base/mfactory",function(S,MFctory,Model){
                var MF=MFctory.create('test/mf',Model);
                MF.registerModels([
                    {
                        type:'Home_List',
                        ops:{
                            uri:'test'
                        },
                        gets:{
                            a:'12'
                        },
                        cacheKey:'',
                        expires:20000,//缓存多久
                        before:function(m){
                            
                        },
                        after:function(m){
                            
                        }
                    },
                    {
                        type:'Home_List1',
                        ops:{
                            uri:'test'
                        },
                        before:function(m){
                            
                        },
                        after:function(m){
                            
                        }
                    }
                ]);
                return MF;
            },{
                requires:["mxext/modelfactory","app/base/model"]
            });

            //使用

                    type:MF.Home_List1,gets:{a:'b'}
                }],function(m1,m2){
    
                },function(msg){
    
                });
            });
